## Bugs & Issues Encountered:

### Issue 1: Upgrade reports success but version unchanged
**Date**: March 23, 2026
**Error**: `aicontext upgrade` prints "Upgrade complete!" followed by "Update available: v1.4.0 → v1.5.0". Running `aicontext version` still shows v1.4.0.
**Root Cause**: (1) Version cache not cleared after upgrade — stale data persists. (2) `checkForUpdates()` always runs after upgrade, comparing old in-memory VERSION against cached latest. (3) No post-install verification — success is assumed if `execSync` doesn't throw.
**Solution**: Clear cache, skip update check, verify installed version after upgrade.
**Prevention**: Any CLI self-upgrade should verify the outcome, not just the exit code.
