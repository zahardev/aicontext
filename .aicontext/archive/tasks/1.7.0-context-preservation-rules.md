# Task 1.7.0: Context preservation rules

## Created:
April 7, 2026

## Spec:
[spec-1.7.0-framework-improvements.md](../specs/spec-1.7.0-framework-improvements.md)

## Objective:
Establish information density (conciseness without information loss) as a project-wide standard. Make sure subagents inherit it. Fix the reviewer-invocation pattern that was overriding the agent's terse defaults. Pivot Question Pacing from atomic-by-default to batch-by-default for independent questions, since round trips are a major hidden context cost.

## Requirements:
