# Brainstorm

Review the current discussion, plan, or approach. Generate ideas we haven't considered:

- **Missing angles** — what haven't we thought about?
- **Better implementations** — simpler, faster, or more maintainable approaches
- **Combinations** — can existing ideas be merged or extended?
- **Alternatives** — completely different ways to solve the same problem

Be generative, not critical — this is about expanding options, not finding flaws.
Present each idea briefly with enough context to evaluate it. Don't filter too aggressively — a rough idea the user can build on is valuable.
