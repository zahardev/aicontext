# Next Step

Advance the active task methodically.

## Workflow

1. Read the active task file and identify the most recently completed step.
2. Update the task file to mark completed checklist items as done when the completed work is clearly reflected in the code or prior context.
3. Update any in-memory or explicit plan so the completed step is marked complete.
4. Check whether recent discoveries should change the task notes, implementation order, or scope.
5. If the plan still holds, begin the next concrete implementation step instead of stopping at planning.
6. If the plan needs adjustment, revise it before continuing.
