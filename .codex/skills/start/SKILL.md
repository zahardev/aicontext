# Start

Inspect the current project just enough to confirm readiness.

## Workflow

1. Check the repository root for obvious project signals such as key config files, package manifests, or task files.
2. Note any immediate blockers only if they are obvious from a quick inspection.
3. Reply in exactly one sentence confirming readiness, or in one sentence stating the most important blocker.
